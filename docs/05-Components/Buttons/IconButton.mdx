## Variants

| Type | Purpose |
| ---- | ------- |
| **Primary** |  If there is no primary action, all Button components should be secondary. |
| **Secondary** | The secondary Button is the default button style and can trigger various actions. When there is no main important action on a given page, all Button components should use the secondary style. |
| **Destructive**| |
| **Disabled** | If the functionality the IconButton provides is not available but the IconButton is supposed to show that there is this kind of functionality.|
